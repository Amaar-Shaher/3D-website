# Keen Comply — 3D Website

الموقع التسويقي التفاعلي لمنصة **Keen Comply** لأتمتة الحوكمة وإدارة المخاطر والامتثال (GRC) — عربي RTL بالكامل، بدون أي مكتبات خارجية (محرك جسيمات 3D مكتوب من الصفر)، وخطوط مضمّنة ذاتياً.

## الصفحات

| المسار | الوصف |
|---|---|
| [`/`](index.html) | الموقع الرئيسي — هيرو 3D، مستكشف الأنظمة العشرة التفاعلي، اللوائح الحية، حاسبة التوفير |
| [`/v3/`](v3/index.html) | نسخة الموقع التفاعلية مع السكرول — خلفية متبدلة، بارالاكس، خطوات مثبّتة |
| [`/systems/`](systems/index.html) | **رحلة الأنظمة العشرة** — Scrollytelling من 12 فصلاً، منحوتة جسيمات وبطاقة قراءات متحركة لكل نظام |
| [`/journey/`](journey/index.html) | رحلة الامتثال الغامرة — الهوية الفاتحة |
| [`/experience/`](experience/index.html) | رحلة الامتثال الغامرة — النسخة الداكنة |

## النشر

**GitHub Pages**: Settings ← Pages ← Deploy from a branch ← `main` / root — ثم:
`https://amaar-shaher.github.io/3D-website/`

أو افتح `index.html` مباشرة في المتصفح — كل شيء يعمل محلياً بلا خادم.
